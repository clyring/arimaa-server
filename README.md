# arimaa-server
[playArimaa](http://playarimaa.org)

## Getting started

### Setup
1. Install [sbt](http://www.scala-sbt.org/download.html) and [nodejs](https://nodejs.org/)
2. `npm install`
3. `gulp build`

### Running
1. `$> sbt`
2. `~container:start`
3. [Go to site](http://localhost:8080)
